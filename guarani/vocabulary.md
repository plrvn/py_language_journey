# The 50 Most Essential Guaraní Words

| Guaraní Word | English Meaning | Usage / Context |
| :--- | :--- | :--- |
| **Rohayhu** | I love you | The most famous phrase. Often used in music. |
| **Ma’erã?** | What for? | Used constantly when questioning a purpose. |
| **Mba’éichapa?** | How are you? | The standard greeting. |
| **Iporã** | It's good / I'm good | The standard response to greetings. |
| **Iporãiterei** | Very good / Excellent | Adding "-iterei" makes any adjective "very." |
| **Aguyje** | Thank you | Essential for politeness. |
| **Avave** | Nobody | Used as "No one" or "None." |
| **Mamópa?** | Where? | Used to ask for a location. |
| **Araka’épa?** | When? | Used for time queries. |
| **Mba’épa?** | What? | Used when you didn't hear something. |
| **He’ẽ** | Yes | Often replaces the Spanish "sí." |
| **Nahániri** | No | The formal "no." |
| **Mombe’u** | To tell / Story | "E-mombe’u chéve" (Tell me). |
| **Jajotopata** | We will see each other | Used as "See you later." |
| **Ka’u** | Drunk | "Está ka'u" (He is drunk). Very common. |
| **Tavy** | Crazy / Ignorant | "Ese es tavy" (He's a bit slow/crazy). |
| **Vaka’i** | Corned beef | A staple food in Paraguay. |
| **Cháke!** | Watch out! | An exclamation of warning. |
| **Nde!** | Hey you! / Wow! | Used to grab attention or show surprise. |
| **Vyrorei** | Nonsense / Trifle | Something of little importance. |
| **Añamemby** | Son of a devil | A common (but strong) insult/exclamation. |
| **Haku** | It is hot | Essential for the Paraguayan weather. |
| **Ro’y** | It is cold | Used during the short winter. |
| **Kuerái** | Fed up / Tired of | "Estoy kuerái" (I've had enough). |
| **Piko?** | (Question marker) | Added to Spanish: "¿Te vas piko?" (Are you going?). |
| **Ra’e** | (Past marker) | Used for realization: "Ah, vos ra'e" (Ah, it was you). |
| **Niko / Ndaje** | (Emphasis markers) | "Niko" is for certainty; "Ndaje" is for hearsay/rumors. |
| **Japo** | To do / make | "Ejapo" (Do it). |
| **Ke** | To sleep | "Eke" (Go to sleep). |
| **Karai** | Sir / Gentleman | Respectful term for a man. |
| **Kuñakarai** | Ma'am / Lady | Respectful term for a woman. |
| **Mitã** | Child / Kid | Often used as "Mitã'i" (little boy). |
| **Kakuaa** | Big / Adult | Also used for someone tall. |
| **Pirite** | Very thin | Used to describe someone skinny. |
| **Kyra** | Fat | Descriptive adjective. |
| **Porã** | Pretty / Beautiful | Can describe people or things. |
| **Vai** | Ugly / Bad | Opposite of Porã. |
| **Péicha** | Like that / This way | "Péicha katu" (That's how it is). |
| **Ápe** | Here | Used for location. |
| **Amo** | There (far) | Used for distant locations. |
| **Ho’u** | To eat / drink | "E'u" (Eat/Drink it). |
| **Yuky** | Salty / Charming | "Qué yuky" (How charming/cute). |
| **Kachi’ãi** | Messy / Joker | Someone who is a bit of a prankster. |
| **Angu’a** | Mortar | The wooden tool used to crush herbs for tereré. |
| **Pohã Ñana** | Medicinal herbs | The herbs added to the tereré water. |
| **Y** | Water | Pronounced as a guttural "i." |
| **Tataypy** | Kitchen/Fireplace | Where the family gathers. |
| **Kosina** | Kitchen | The Jopara (hispanized) version. |
| **Tape** | Path / Road | Used for streets or ways. |
| **Anichéne!** | No way! | An exclamation of disbelief. |
