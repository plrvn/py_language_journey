# Essential Jopara Phrases (Spanish-Guaraní Mix)

| Phrase | Literal/Component Meaning | English Equivalent |
| :--- | :--- | :--- |
| **¿Mba'éichapa el día?** | (G) How are you + (S) the day? | How is your day going? |
| **¡Haku katu!** | (G) It is hot + (G) intensity marker | It is incredibly hot! |
| **¿Mba'épa la porte?** | (G) What is + (S) the bearing/style? | What's up? / How's it hanging? |
| **Tranquilo pa?** | (S) Calm + (G) question marker | Is everything cool? |
| **¡Anichéne, en serio!** | (G) No way + (S) seriously | No way, are you serious?! |
| **¡Cháke el jagua!** | (G) Watch out + (S) the + (G) dog | Watch out for the dog! |
| **¿Mba'e piko eso?** | (G) What + (G) question marker + (S) that? | What on earth is that? |
| **¡Iporã la vida!** | (G) Is good + (S) the life | Life is good! |
| **¡Eju lunes!** | (G) Come + (S) Monday | Come on Monday! (Common for appointments) |
| **¡Vení katu!** | (S) Come + (G) emphasis/invitation | Come here already! |
| **¿Entendés piko?** | (S) Do you understand + (G) question marker | Do you actually understand? |
| **¡Qué vyrorei!** | (S) What + (G) nonsense/stupid thing | What a load of nonsense! |
| **Estoy kuerái de esto.** | (S) I am + (G) fed up + (S) of this | I am totally fed up with this. |
| **Él es un kachi'ãi.** | (S) He is a + (G) joker/messy person | He's such a prankster/unreliable. |
| **¿Querés tereré ra'e?** | (S) You want tereré + (G) past/realization | So, did you want tereré (after all)? |
| **¡Ndéve guarã!** | (G) For you | This is for you! (Often used as "Take that!") |
| **No hay avave.** | (S) There is no + (G) nobody | There is nobody here. |
| **¡Dios se lo pague, aguyje!** | (S) God pay you + (G) thanks | Thank you very much, God bless! |
| **¡Qué tavy es!** | (S) How + (G) crazy/slow + (S) he is | He's so clueless! |
| **¡Mba'eteko, kape!** | (G) What's the status + (S) slang for friend | What's up, buddy? |
