# Paraguayan Spanish Slang (Paraguayismos)

| Slang Word | Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Purete** | Cool / Great | "¡Qué purete tu auto!" (Your car is so cool!) |
| **Chuchi** | Fancy / Poshy | Used to describe expensive places or snobbish people. |
| **Bolonqui** | A mess / Problem | Backwards slang for "Quilombo." "Se armó el bolonqui." |
| **Kape** | Friend / Buddy | From "capitán." Used like "bro" or "dude." |
| **Valecho** | Skilled / Capable | Someone who is very good at what they do. |
| **Tova Atã** | Shameless | Literally "Hard face." Someone with no shame. |
| **Cachada** | A joke / Prank | "Es una cachada" (It's a joke/not serious). |
| **Tembi'u** | Food | Often used in Spanish sentences to refer to a meal. |
| **Gua'u** | Fake / "Just kidding" | "De gua'u nomás" (I'm just kidding / It's just for show). |
| **A mil** | Very fast / Busy | "Estoy a mil" (I'm super busy/rushing). |
| **Bolsicuaco** | Pickpocket | A local term for a petty thief. |
| **Chulina** | How cute! | An exclamation used when seeing a baby or a puppy. |
| **Socorro** | Help / Wow | Used as an exclamation of shock or "Oh my God!" |
| **Legalmente** | Truthfully / Honestly | Used as a filler word: "Legalmente no sé" (Honestly, I don't know). |
| **Hallarse** | To feel at home | "No me hallo" (I don't feel comfortable/I don't fit in here). |
| **Plata mbytére** | Extremely rich | Used to describe someone with "middle-of-the-money" (tons of it). |
| **Hule** | Done for / Game over | "Ya está hule" (It's over / It's ruined / He's dead). |
| **Nervios** | To be angry | In Paraguay, "Estoy con nervios" means "I am angry," not "I am nervous." |
| **Médico a palos** | Unqualified person | Someone pretending to be an expert in something they aren't. |
| **Ondita** | Good vibes | "Tiene ondita" (It has a cool vibe). |
